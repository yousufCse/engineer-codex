Physical, Data Link, Network, Transport, Session, Presentation, Application — প্রতিটি লেয়ারের কাজ, পারস্পরিক সম্পর্ক। Data encapsulation ও decapsulation। একটি HTTP রিকোয়েস্ট কিভাবে ৭ লেয়ার পার হয় — বিস্তারিত ডায়াগ্রাম সহ।
OSI 7 Layers Encapsulation Protocol Stack

TCP/IP মডেল — ইন্টারনেটের আসল ভিত্তি
TCP/IP এর ৪টি লেয়ার বিস্তারিত। OSI vs TCP/IP তুলনা। প্রতিটি লেয়ারে কোন প্রোটোকলগুলো কাজ করে। Flutter অ্যাপ থেকে সার্ভার পর্যন্ত একটি রিকোয়েস্টের সম্পূর্ণ যাত্রা।

ফিজিক্যাল মিডিয়া ও সিগন্যাল ট্রান্সমিশন
Ethernet cable (Cat5e/Cat6/Cat7), Fiber Optic, Wi-Fi। সিগন্যাল কোডিং: Manchester, NRZ। Error detection: Parity, Checksum, CRC। কেন ফাইবার ওয়্যার বেশি দ্রুত? Wi-Fi কিভাবে কাজ করে?
Ethernet Fiber Wi-Fi Signal Coding
🏷️ CH 07
MAC Address ও Ethernet ফ্রেম
MAC Address কী, কোথায় থাকে, কিভাবে কাজ করে। Ethernet Frame গঠন: Preamble, Destination, Source, EtherType, Payload, FCS। ARP (Address Resolution Protocol) — IP থেকে MAC কিভাবে বের হয়। Flutter ডিভাইসের MAC Address।
MAC Address Ethernet Frame ARP
🔀 CH 08
Switch, Hub ও Bridging
Hub vs Switch vs Router — বিস্তারিত পার্থক্য। Switch কিভাবে MAC Table তৈরি করে। Spanning Tree Protocol (STP)। VLAN (Virtual LAN) — একটি অফিসে বিভাগ আলাদা করা। Collision Domain vs Broadcast Domain।
Switch VLAN STP MAC Table

🔢 CH 09
IP Address — IPv4 ও IPv6 বিস্তারিত
IPv4 গঠন ও Classes (A/B/C/D/E)। Private vs Public IP। Special IPs: Loopback (127.0.0.1), Broadcast। IPv6 কেন দরকার হলো, এর গঠন। Unicast, Multicast, Anycast, Broadcast। আপনার Flutter অ্যাপ কোন IP থেকে কথা বলে?
IPv4 IPv6 Public/Private IP Loopback
✂️ CH 10
Subnetting ও CIDR — নেটওয়ার্ক ভাগ করার কৌশল
Subnet Mask কী এবং কেন দরকার। CIDR Notation (/24, /16, /8)। Subnetting calculation বিস্তারিত উদাহরণ সহ। VLSM (Variable Length Subnet Masking)। AWS/GCP VPC-তে কিভাবে subnet ডিজাইন করবেন।
Subnet Mask CIDR VLSM VPC Design
🗺️ CH 11
Routing — ডেটা প্যাকেটের পথ নির্ধারণ
Static vs Dynamic Routing। Routing Table কিভাবে কাজ করে। Routing Protocols: RIP, OSPF, BGP — কোনটি কখন। BGP — ইন্টারনেটের মেরুদণ্ড। Traceroute দিয়ে packet path দেখা। Longest Prefix Match।
Routing Table OSPF BGP Traceroute
🔄 CH 12
NAT ও PAT — একটি IP দিয়ে অনেক ডিভাইস
NAT (Network Address Translation) কেন দরকার। Static NAT, Dynamic NAT, PAT (Port Address Translation)। আপনার বাসার রাউটার কিভাবে NAT করে। Mobile নেটওয়ার্কে Carrier-Grade NAT। Flutter অ্যাপ ও NAT traversal সমস্যা।
NAT PAT Port Forwarding CGNAT

🤝 CH 13
TCP — নির্ভরযোগ্য যোগাযোগের বিজ্ঞান
TCP 3-way Handshake (SYN, SYN-ACK, ACK) বিস্তারিত। 4-way Teardown। Flow Control ও Congestion Control। Sliding Window Protocol। TCP States: ESTABLISHED, TIME_WAIT, CLOSE_WAIT। Retransmission ও Timeout। Flutter HTTP call কিভাবে TCP ব্যবহার করে।
3-way Handshake Flow Control Congestion Control Sliding Window
⚡ CH 14
UDP — দ্রুত কিন্তু অনিশ্চিত
UDP কেন connectionless? কখন UDP ব্যবহার করবেন। DNS, Video Streaming, Online Gaming, VoIP — কেন UDP? QUIC প্রোটোকল (UDP এর উপর নির্মিত HTTP/3)। Flutter-এ UDP socket ব্যবহার।
UDP QUIC HTTP/3 Streaming
🚪 CH 15
Port ও Socket — যোগাযোগের দরজা
Port Number কী (0-65535)। Well-known Ports: 80, 443, 22, 3306, 5432। Ephemeral Ports। Socket কী — IP + Port = Socket। Socket Programming মূলধারণা। Flutter-এ TCP Socket সংযোগ। Port scanning ও নিরাপত্তা।
Port Numbers Socket Ephemeral Port Socket Programming

DNS কিভাবে কাজ করে (Recursive vs Iterative Query)। DNS Record Types: A, AAAA, CNAME, MX, TXT, NS, PTR। DNS Hierarchy: Root → TLD → Authoritative। DNS Caching ও TTL। DoH (DNS over HTTPS), DoT। Flutter অ্যাপে DNS lookup। DNS Attack: Cache Poisoning, DDoS।

HTTP/HTTPS — ওয়েবের প্রাণ
HTTP/1.0 → HTTP/1.1 → HTTP/2 → HTTP/3 বিবর্তন। Request/Response গঠন বিস্তারিত। Methods: GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS। Status Codes সম্পূর্ণ তালিকা ও অর্থ। Headers: Content-Type, Authorization, Cache-Control, CORS। Keep-Alive, Pipelining, Multiplexing। Flutter Dio/http প্যাকেজ ও HTTP internals।
HTTP Methods Status Codes HTTP/2 HTTP/3 CORS

HTTP vs WebSocket পার্থক্য। WebSocket Handshake বিস্তারিত। Full-duplex communication। Use cases: Chat, Live Notification, Stock Price, Multiplayer Game। Server-Sent Events (SSE) কখন ব্যবহার করবেন। Flutter-এ WebSocket implementation। Scaling WebSocket: Sticky Sessions, Redis Pub/Sub।
WebSocket Real-time SSE Full-duplex

REST vs gRPC — কোনটি কখন? Protocol Buffers (Protobuf) বিস্তারিত। 4 ধরনের gRPC: Unary, Server Streaming, Client Streaming, Bidirectional। HTTP/2 এর উপর gRPC। Flutter-এ grpc package ব্যবহার। gRPC vs GraphQL vs REST তুলনা।
