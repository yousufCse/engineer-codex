> A complete knowledge vault for Software Engineers — covering LLD, Flutter Interviews, Networking, Books & more.

## 🎯 What is Engineer Codex?

**Engineer Codex** is a personal knowledge base and resource collection built for Software Engineers who want to **learn, prepare and grow** in their career.

Whether you are preparing for interviews, learning system design, or reading software engineering books — this is your **one stop resource hub**.
